## Getting Started

Welcome to the VS Code Java world. Here is a guideline to help you get started to write Java code in Visual Studio Code.

## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies

Meanwhile, the compiled output files will be generated in the `bin` folder by default.

> If you want to customize the folder structure, open `.vscode/settings.json` and update the related settings there.

## Dependency Management

The `JAVA PROJECTS` view allows you to manage your dependencies. More details can be found [here](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).

## Задача 1. Площадь прямоугольника

Напишите программу для вычисления площади прямоугольника по заданным длине и ширине.

## Задача 2. Проверка на четность числа

Напишите программу, которая проверяет, является ли введенное число четным или нечетным.

## Задача 4. Перевод из градусов Цельсия в градусы Фаренгейта

Напишите программу, которая принимает температуру в градусах Цельсия и выводит ее эквивалент в градусах Фаренгейта.

## Задача 5. Поиск максимального числа в массиве

Напишите программу для поиска максимального числа в заданном массиве.

## Задача 6. Перевернуть строку

Напишите программу для обращения введенной строки задом наперед.

## Задача 7. Проверка на палиндром

## Задача 8. Генерация случайного числа и угадывание

Напишите игру, в которой компьютер загадывает случайное число, а пользователь пытается угадать его.

## Задача 9. Подсчет суммы чисел в интервале

Напишите программу, которая считает сумму всех чисел в заданном пользователем интервале.

## Задача 13. Перевод из десятичной системы счисления в двоичную

Напишите программу для перевода введенного числа из десятичной системы счисления в двоичную.

Задача 15. Обратный порядок слов в предложении
